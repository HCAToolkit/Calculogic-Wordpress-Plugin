== Description ==
Calculogic is an advanced drag-and-drop form builder plugin for WordPress, designed to revolutionize the way forms are created and managed. With a user-friendly visual editor, users can effortlessly build forms with advanced field types, dynamic calculations, real-time feedback, and more.

== Installation ==
1. Upload the `calculogic-wordpress-plugin` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Use the Calculogic menu in the admin panel to start creating forms.

== Features ==
- Drag-and-Drop Interface: A user-friendly visual editor for building forms effortlessly.
- Advanced Field Types: Includes calculated fields, repeatable fields, and custom HTML.
- Dynamic Calculations and Logic: Embed complex arithmetic and logic directly within forms.
- Real-Time Feedback: Immediate validation and error messages during form filling.
- Collaboration Tools: Support for real-time editing and user permissions management.
- Backend Integration: Synchronization with databases and external APIs.
- Security and Compliance: Comprehensive security features to prioritize data integrity and privacy.

== Frequently Asked Questions ==
= How do I create a form? =
Navigate to the Calculogic menu in the admin panel and follow the prompts to build your form.

= Can I customize the form styles? =
Yes, you can add custom CSS in the plugin settings to style your forms.
